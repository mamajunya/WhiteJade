# WhiteJade 使用指南

## 快速上手（3步）

### 第一步：启动程序
双击 `启动GUI.bat` 或运行 `python gui_app.py`

### 第二步：获取 Token
1. 切换到"Token 获取"标签页
2. 输入 Pixiv 账号和密码
3. 点击"获取 Token"
4. 点击"保存到配置"

### 第三步：开始下载
1. 切换到"下载与过滤"标签页
2. 输入搜索关键词（如：初音未来）
3. 设置下载数量（如：20）
4. 点击"开始执行"

## 详细设置

### 下载设置

| 选项 | 说明 | 推荐值 |
|------|------|--------|
| 搜索关键词 | 角色名或标签 | 例如：铃兰 |
| 下载数量 | 最终保留的图片数 | 20-50 |
| 最小收藏 | 过滤低质量作品 | 100-1000 |
| 跳过 R-18 | 过滤成人内容 | ✓ 建议勾选 |
| 去除 AI 作品 | 过滤 AI 生成 | ✓ 建议勾选 |

### 审核设置

| 选项 | 说明 | 推荐值 |
|------|------|--------|
| 启用图片审核 | 自动过滤不适宜内容 | ✓ 建议启用 |
| 检测阈值 | 审核严格程度 | 0.4（非常严格） |
| 移动到 ban | 保留被过滤的图片 | ✓ 建议勾选 |

### 阈值选择指南

- **0.4 - 非常严格**：适合公开场合，过滤最严格 ⭐推荐
- **0.5 - 严格**：较严格，适合一般使用
- **0.6 - 默认**：平衡模式
- **0.7 - 宽松**：较宽松
- **0.8 - 非常宽松**：最宽松

## 任务控制

### 开始执行
点击"开始执行"按钮启动任务

### 暂停任务
点击"暂停"按钮暂停当前任务，可随时继续

### 继续任务
暂停后点击"继续"按钮恢复任务

### 停止任务
点击"停止"按钮完全停止任务（需确认）

## 查看结果

### 下载位置
```
downloads/
└── [关键词]/
    ├── picture/    ← 合格图片在这里
    ├── ban/        ← 被过滤图片在这里
    └── history/    ← 下载历史
```

### 日志说明
程序会显示详细日志：
- `[登录]` - 登录相关信息
- `[下载]` - 下载进度
- `[审核]` - 审核进度
- `[控制]` - 任务控制操作
- `[完成]` - 任务完成信息

## 故障排除

### 问题：登录失败
**解决方案**：
1. 检查账号密码是否正确
2. 重新获取 Token
3. 检查网络连接

### 问题：下载速度慢
**解决方案**：
1. 检查网络连接
2. 降低下载数量
3. 使用代理（如需要）

### 问题：审核太严格
**解决方案**：
1. 提高检测阈值（0.5-0.6）
2. 或关闭审核功能

### 问题：保留图片不够
**解决方案**：
1. 降低最小收藏数
2. 提高检测阈值
3. 更换关键词

### 问题：首次运行很慢
**原因**：正在下载 DeepDanbooru 模型（600MB）
**解决方案**：耐心等待，只需下载一次

## 高级技巧

### 技巧1：批量下载多个角色
分别输入不同关键词，多次执行

### 技巧2：查看过滤原因
查看 `history/` 目录中的 JSON 文件

### 技巧3：自定义主题
在"设置"标签页选择喜欢的主题色

### 技巧4：恢复被过滤图片
从 `ban/` 目录移回 `picture/` 目录

## 注意事项

1. ⚠️ 首次运行需要下载模型，约 600MB
2. ⚠️ Token 会过期，需定期重新获取
3. ⚠️ 下载数量越大，耗时越长
4. ⚠️ 审核会增加处理时间
5. ⚠️ 确保有足够的磁盘空间
