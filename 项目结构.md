# WhiteJade 项目结构

```
WhiteJade/
│
├── 📄 启动GUI.bat              # Windows 一键启动脚本
├── 📄 gui_app.py               # GUI 主程序
├── 📄 environment_checker.py   # 环境检测模块
├── 📄 environment_dialog.py    # 环境检测对话框
├── 📄 icon.png                 # 应用图标
│
├── 📖 README.md                # 项目说明文档
├── 📖 使用指南.md              # 详细使用教程
├── 📖 CHANGELOG.md             # 更新日志
├── 📖 项目结构.md              # 本文件
│
├── 📋 requirements.txt         # Python 依赖列表
├── 📋 .gitignore              # Git 忽略配置
│
├── 📁 pixiv_downloader/        # Pixiv 下载器模块
│   ├── pixiv_downloader.py    # 下载器核心代码
│   ├── config.json            # 配置文件（Token等）
│   ├── config.example.json    # 配置文件示例
│   ├── requirements.txt       # 下载器依赖
│   └── .gitignore            # 下载器忽略配置
│
├── 📁 image_moderator/         # 图片审核模块
│   ├── deepdanbooru_moderator.py  # DeepDanbooru 审核器
│   ├── requirements.txt           # 审核器依赖
│   ├── requirements_deepdanbooru.txt  # DeepDanbooru 依赖
│   ├── README.md                  # 审核器说明
│   └── .gitignore                # 审核器忽略配置
│
├── 📁 model/                   # DeepDanbooru 模型文件
│   ├── model-resnet_custom_v3.h5  # 模型权重（617MB）
│   ├── tags.txt                   # 标签列表（9176个）
│   ├── tags-general.txt           # 通用标签
│   ├── tags-character.txt         # 角色标签
│   ├── project.json               # 模型配置
│   ├── categories.json            # 分类配置
│   └── tags_log.json             # 标签日志
│
└── 📁 downloads/               # 下载根目录（自动创建）
    └── [关键词]/              # 按关键词分类
        ├── picture/           # ✅ 合格图片
        ├── ban/              # ❌ 被过滤图片
        └── history/          # 📝 下载历史和元数据
            ├── [ID]_info.json        # 作品信息
            └── download_history.json # 下载历史
```

## 核心文件说明

### 主程序
- **gui_app.py**: GUI 主程序，包含所有界面逻辑和任务控制
- **启动GUI.bat**: Windows 快速启动脚本

### 环境管理
- **environment_checker.py**: 检测并安装 Python 环境和依赖
- **environment_dialog.py**: 环境检测进度对话框

### 下载模块
- **pixiv_downloader.py**: 
  - Pixiv API 封装
  - 搜索和下载功能
  - 历史记录管理
  - 去重机制

### 审核模块
- **deepdanbooru_moderator.py**:
  - DeepDanbooru 模型加载
  - 图片内容分析
  - 过滤标签检测
  - 批量处理功能

### 配置文件
- **config.json**: 存储 Pixiv Token 和用户设置
- **config.example.json**: 配置文件模板

### 模型文件
- **model-resnet_custom_v3.h5**: 深度学习模型权重
- **tags.txt**: 完整标签列表（9176个）
- **project.json**: 模型配置参数

## 数据流程

```
用户输入
    ↓
GUI 界面 (gui_app.py)
    ↓
WorkThread 工作线程
    ↓
┌─────────────────┐
│  下载阶段        │
│  PixivDownloader │
│  - 搜索作品      │
│  - 下载图片      │
│  - 保存元数据    │
└─────────────────┘
    ↓
┌─────────────────┐
│  审核阶段        │
│  DeepDanbooru    │
│  - 加载模型      │
│  - 分析图片      │
│  - 过滤内容      │
└─────────────────┘
    ↓
结果输出
    ↓
picture/ 或 ban/
```

## 目录权限

### 可修改
- `downloads/` - 下载的图片
- `pixiv_downloader/config.json` - 配置文件

### 不建议修改
- `model/` - 模型文件
- `pixiv_downloader/` - 核心代码
- `image_moderator/` - 核心代码

### 自动生成
- `__pycache__/` - Python 缓存
- `downloads/` - 下载目录
- `temp_get_token.ps1` - 临时脚本

## 依赖关系

```
gui_app.py
├── PyQt6 (GUI框架)
├── pixiv_downloader.py
│   ├── pixivpy3 (Pixiv API)
│   └── gppt (Token获取)
└── deepdanbooru_moderator.py
    ├── deepdanbooru (AI模型)
    ├── tensorflow (深度学习)
    └── Pillow (图片处理)
```

## 存储空间需求

| 项目 | 大小 | 说明 |
|------|------|------|
| 程序代码 | ~5 MB | Python 脚本 |
| DeepDanbooru 模型 | ~617 MB | 一次性下载 |
| 下载图片 | 变动 | 取决于下载数量 |
| Python 环境 | ~500 MB | 如需自动安装 |

**建议预留空间**: 2-5 GB

## 网络要求

- Pixiv 服务器访问（日本）
- DeepDanbooru 模型下载（首次）
- Python 包下载（首次）

**建议**: 稳定的网络连接，首次运行可能需要较长时间
