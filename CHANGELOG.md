# 更新日志

## [2.0.0] - 2024-12-XX

### 新增功能
- ✨ 全新 PyQt6 GUI 界面
- 🤖 集成 DeepDanbooru AI 内容审核系统
- 🎯 智能数量控制：自动调整下载量确保达到目标数量
- ⏸️ 任务控制：支持暂停、继续、停止操作
- 🎨 主题定制：可自定义界面主题色和背景色
- 🌍 多语言支持：中文、日语、韩语、英语、法语、德语
- 🚫 AI 作品过滤：自动跳过 AI 生成的作品
- 📊 详细日志：实时显示下载和审核进度
- 🔄 自动环境检测：首次运行自动安装依赖

### 改进优化
- 📁 优化目录结构：picture/ban/history 三级分类
- 🔍 增强筛选条件：支持最小收藏数、R-18 过滤、AI 过滤
- 📈 智能下载策略：根据过滤率动态调整下载量
- 🎚️ 可调审核阈值：5 档严格度可选（0.4-0.8）
- 💾 自动去重：避免重复下载相同作品
- 📝 详细统计：显示下载数、审核数、过滤率等

### 技术更新
- 🔧 使用 DeepDanbooru 替代 NudeNet
- 🔧 优化 Token 获取流程
- 🔧 改进错误处理和日志输出
- 🔧 移除表情符号，使用标签化日志
